# 2026-06-26 工作日志

## 修复 @ant-design/icons v6 图标不兼容错误

**问题**：项目构建报 `Attempted import error: 'XXX' is not exported from '@ant-design/icons'` 系列错误，约 20+ 个图标名称在 v6 版本中不存在。

**根因**：项目组件的图标导入使用了旧版本或不存在的图标名称，与已安装的 @ant-design/icons v6.3.0 不兼容。

**修复**：将所有缺失图标替换为 v6 中存在的等价图标（详见 mapping）。
- 涉及文件：Dashboard.jsx, Header.jsx, Sidebar.jsx, Settings.jsx, Login.jsx, RadioPlayer.jsx, Minesweeper.jsx, FaceCapture.jsx, FaceVerify.jsx, AdminFaceManager.jsx, UserManagement.jsx, SeatManagement.jsx, ReservationManagement.jsx
- 构建结果：Compiled with warnings（无错误），ESLint 仅有 unused-vars 警告

**主要替换映射**：
TrashOutlined→DeleteOutlined, XOutlined/BanOutlined→CloseOutlined/StopOutlined, BombOutlined→FireOutlined, BuildingOutlined→BankOutlined, AlertCircleOutlined→ExclamationCircleOutlined, ShieldCheckOutlined→SafetyOutlined, 等
